### Q1: Which point(s) were new to you?
Companies use some tools for documentation like Trello, Jira, but if we are not using any tool we should write down the requirements and share it with team members.

# Which area do you think you need to improve on? What are your ideas to make progress in that area?
I should work on blocking social media apps, or stop using for a long time during work hours. Even there are some tools available for this like TimeLimit, Freedom, and many more 
- Keep my phone in another room or enable Do Not Disturb while working.
- Use the Pomodoro Technique (25–50 minutes of focused work followed by a short break).
- Set specific times to check messages or social media instead of checking them frequently.
- Create a daily task list and prioritize the most important tasks first.
- Turn off non-essential notifications on my phone and laptop.
